# Atari-2600-Bomber
Top scroller - Shooter game similar to "River Raider" on the vintage Atari 2600 system.

Key commands:
left and right arrow keys:     Move the drone left and right
spacebar:                      Fire
Escape:                        Quit

# Playing

Install Stella from here: https://stella-emu.github.io/

The .asm and .h files aren't necessary to run. Open Stella, navigate to your folder with cart.bin (this is the binary file acting like a game cartridge), and open it.

The game is designed to run on NTSC format. To ensure, open Stella, under Options->Game Properties, under TV Format, select NTSC.

Run and enjoy!
